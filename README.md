| **Design** | Système de design unifié (dark mode noir/or + thèmes secteurs) |
| **Icons** | Emojis natifs (compatibilité universelle, zero dépendance) |
| **Responsive** | Mobile-first, breakpoints à 600px / 800px / 980px |
